# wordpress-wordfence-blockings-log

Parse log generated by the Wordpress plugin [Wordfence blockings log](https://github.com/Cyrille37/wordfence-blockings-log).
Log contains Wordfence throttles and block events.

## acquistion configuration

 ```yaml
 filenames:
  - /opt/wordpress/wp-content/logs/wordfence_security_event.log
 labels:
   type: wordpress-wordfence-blockings
 ```
